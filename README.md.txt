# Infinite Fall

ブラウザで遊べる、落下型ローグライク風ゲームです。  
プレイヤーはひたすら落下しながら、敵と戦い、生き延びます。
昔HyperCardで遊んだ落下ゲームが今頃遊びたくなり、ClaudeとGPT頼みで再現してみました。

🔽 プレイはこちら  
https://rubedo108.github.io/Infinite-Fall/

---

## 🎮 ゲーム概要

- 自動で2mずつ落下します。落下しているとお腹がすきます…。
- 満腹度が0になると死亡。
- 戦闘中も落下は止まらず落ち続けます。
- 敵を倒すと経験値が貯まり回復薬かごはんをランダム入手します。
- 装備の入手は自動かつランダムです。
- アイテムを捨てるとHPと満腹度が回復する謎仕様です。上手に断捨離して回復して落下してください。
- 深く潜るほど敵は強くなります。深淵の底に待つものは…。

---

## 🕹 操作方法

### ボタン操作
- **スタート**：落下開始
- **攻撃する**：敵を攻撃
- **逃げる**：戦闘から逃走（確率）
- **最初から**：リスタート

### キーボード
- **Enter / Space**：攻撃 or 開始
- **X**：逃げる

※ スマホ・PC 両対応
